COVID-19 Report Generator

This is a natural language report generator that generates a short snapshot report of COVID-19 given a specific location and date.

The goal of this generator is help people better understand the situation of COVID-19 and take proper measures under different circumstances. By this generator, people can simply spend a few minitues reading the report instead of getting lost in enormous data.

This generator includes two parts, a data processor and a natural language processor. The data processor is responsible for pre-processing of data related to COVID-19. It first collects data from public COVID-19 dataset like New York Times Coronavirus Dataset, then processes data according to different algorithms, and finally stores results as csv files.

The natural language processor is reponsible for generating a four-paragraphs report based on the processed data. The generated report includes tendency, comparation, suggestions, facts and basically everything a person need in the pandemic.
